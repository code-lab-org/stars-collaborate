Determines if a line-of-sight path exists between two positions.

Parameters
[in]_subjectPosition of subject node
[in]_otherPosition of other node
Returns
Whether a line-of-sight path exists

\[ \vec{s} = subject~position~vector \]

\[ \vec{t} = other~position~vector \]

\[ a = b = EARTH~semimajor~axis,~ c = EARTH~semiminor~axis \]

\[ \vec{q}=\begin{bmatrix} 1 \div a&1 \div b&1 \div c\end{bmatrix} \]

\[ \vec{s}_{q} = \vec{s}\vec{q},~\vec{t}_{q} = \vec{t}\vec{q} \]

\[ \vec{r}_{st} = \vec{s}_q - \vec{t}_{q},~ \vec{r}_{ts} = \vec{t}_q - \vec{s}_{q} \]

\[ Discriminant~of~Quadratic~Formula: \]

\[ d_{st} = (2(\vec{s} \cdot \vec{r}_{st}))^2 - 4(\left\lvert \vec{r}_{st} \right\rvert^2) (\left\lvert\vec{s}\right\rvert^2 - 1) \]

\[ d_{ts} = (2(\vec{t} \cdot \vec{r}_{ts}))^2 - 4(\left\lvert \vec{r}_{ts} \right\rvert^2) (\left\lvert\vec{t}\right\rvert^2 - 1) \]

\[ Visible = (d_{st} > 0) \lor (d_{ts} > 0) \]
